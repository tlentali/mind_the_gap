# Go away

Displays the next tram departure,  
because I don't like watches.

![alt text](https://raw.githubusercontent.com/tlentali/go_away/master/picture/result.png)


### To launch

`./launch.sh`

### Requirement

- requests
- bs4
- re

### Desactivate automatic turn off screen in Gnu/linux

$ xset s off -dpms
